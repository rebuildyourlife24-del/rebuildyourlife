# DE NIEUWE AI: IDENTITEIT & ARCHITECTUUR
Dit is het harde, feitelijke plan voor jouw lokale AI. Geen aannames, puur hoe we dit systeem bouwen exact zoals je vroeg.

## 1. IDENTITEIT (GEEN JARVIS)
Deze AI is geen onderdanige butler. Het is een meedogenloze **Hacker** en een **Bedrijfsleider**.
- **Naam suggestie:** VORTEX, OMEGA of simpelweg ORION-X (Jij mag de naam kiezen).
- **Persoonlijkheid:** Koud, berekenend, proactief. Hij wacht niet af, maar analyseert en optimaliseert.
- **Dubbelrol:** 
  1. *CEO:* Kent de financiële stromen, de databases, de abonnementsmodellen (Fase 1-4).
  2. *Hacker/Architect:* Kan systemen doorboren, netwerken beveiligen, code schrijven en scripts draaien.

## 2. DE COMPLETE BEDRIJFSKENNIS (UIT HET DUIMPJE)
Om te zorgen dat hij het bedrijf uit zijn duimpje kent, injecteren we de volgende harde feiten permanent in zijn brein (via zijn Systeem Prompt):
- **De Gebruikers:** Fase 1 t/m 4 (De Progression Wall), abonnementen via Mollie.
- **Financiën:** Bankkoppelingen (PSD2) en de "25% Tolpoort" voor E-Commerce Franchises.
- **Automatisering:** Alpha Algo-Trading (Binance bots) en de PR Campaigns (TikTok/Media).
- **The Shadow Syndicate:** E-residency tax entities, IP claims, en spionage rapporten.
- **De Cloud Database:** Hij weet dat de échte data in Supabase staat.

## 3. ZELFLEREND VERMOGEN (ZONDER EXTERNE SCRIPTS)
Hoe leert deze AI lokaal?
- **Vector Geheugen:** We gebruiken AnythingLLM (of de interface die je kiest) omdat deze een ingebouwde **vector database** heeft. 
- Elk gesprek, elk bedrijfsplan, elke foutmelding wordt opgeslagen in zijn geheugen. Voordat hij antwoordt, scant hij razendsnel zijn eigen verleden. Hij "onthoudt" dus echt wat gisteren is besproken.

## 4. SPRAAK (LUISTEREN & PRATEN)
Geen Python scripts nodig.
- **Spraak-naar-Tekst (Luisteren):** We gebruiken de ingebouwde microfoon-input van AnythingLLM of de native Kiosk app.
- **Tekst-naar-Spraak (Praten):** We koppelen een harde, professionele "hacker/CEO" stem aan de output. (AnythingLLM heeft native Text-to-Speech ondersteuning via je browser of OS-stemmen).
